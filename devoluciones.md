### HTML

Bien en general. Buena uso de las etiquetas, la estructura y los niveles de títulos.

No usar mayúsculas para contenido, se trata de una decisión de diseño, por lo que debe ser aplicada via CSS. Además puede afectar la accesibilidad, porque ciertos lectores de pantalla pueden tener dificultad para inerpretarlo.
